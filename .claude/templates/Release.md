# Release — ダウンストリームリリース手順テンプレート

> このファイルを `.claude/Release.md` にコピーし、プロジェクトに合わせてカスタマイズしてください。
> `/addf-release` がこのファイルを読み、リリース手順として実行します。

## プレリリースチェック

1. プロジェクトのビルドが通ること
2. テストが全て通過すること
3. `TODO.md` に未完了の Critical タスクがないこと
